## 💻 Пример кода на Python: Генератор надежных паролей

> **Описание задачи:** Написать Python-скрипт для генерации криптографически стойкого пароля, чтобы защититься от атак полного перебора (*brute-force*).

Важное правило, которое я усвоил: для генерации паролей или ключей безопасности **нельзя** использовать стандартную библиотеку `random`, так как она предсказуема. Вместо нее нужно использовать библиотеку `secrets`.

```python
import secrets
import string

def generate_secure_password(length=16):
    """Генерирует криптографически надежный пароль."""
    
    # Собираем алфавит: буквы (верхний и нижний регистр), цифры и спецсимволы
    alphabet = string.ascii_letters + string.digits + string.punctuation
    
    # Генерируем пароль с помощью безопасного модуля secrets
    secure_password = ''.join(secrets.choice(alphabet) for _ in range(length))
    
    return secure_password

# Тестируем функцию
if __name__ == "__main__":
    my_password = generate_secure_password(20)
    print(f"🔑 Сгенерированный пароль : {my_password}")
